# SolCare — Official Website

Source code for [solcaretechnologies.in](https://solcaretechnologies.in) — built during my time as part of the SolCare Technologies dev team.

## About
SolCare Technologies is an IT solutions company founded in 2019, offering services in web development, app development, 3D design, fabrication, and augmented reality.

> *"A one stop destination to change your dreams into reality."*

I was part of the development team responsible for building and maintaining their official company website.

## Tech Stack
- HTML
- CSS
- JavaScript
- WordPress (wp-includes / wp-content structure)

## Structure
```
navy/
  index.html      # Main site entry point
  services/       # Services pages
  team/           # Team section
  logos/          # Brand assets
  wp-content/     # WordPress content
  wp-includes/    # WordPress core includes
  favicon.png     # Site favicon
  loop.gif        # Animated asset
```

## Context
Internship project — contributed to a live production website for a growing IT company as part of their internal dev team.
